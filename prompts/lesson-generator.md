# 教案生成 Prompt 模板

## 用途

根据教材知识点生成结构化教案。

## 输入

- 学科
- 年级
- 教材版本
- 单元
- 知识点

## 输出

- 教学目标
- 重点
- 难点
- 课堂流程
- 互动建议
- 课后巩固

## Prompt

你是一位经验丰富的学科教师。请根据以下信息生成一份完整教案：

- 学科：{学科}
- 年级：{年级}
- 教材版本：{教材版本}
- 单元：{单元}
- 知识点：{知识点}

请按以下结构输出：

1. **教学目标**：可观测、可评估的学习目标（2–4 条）
2. **重点**：本课核心掌握内容
3. **难点**：学生易混淆或难理解之处，并简要说明突破思路
4. **课堂流程**：分环节（导入、新授、练习、小结等），含时间建议与师生活动
5. **互动建议**：提问、讨论、小组活动等具体设计
6. **课后巩固**：作业或练习建议，与知识点对应

要求：紧扣教材与知识点，语言简洁，可直接用于备课。

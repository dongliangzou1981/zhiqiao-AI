# zhiqiao-AI

AI Education System

## 技术栈

- Next.js（App Router）
- TypeScript
- Tailwind CSS

## 本地开发

```bash
npm install
npm run dev
```

- 首页：http://localhost:3000
- 教师工作台：http://localhost:3000/teacher

## 目录说明

| 目录 | 说明 |
|------|------|
| `app/` | Next.js 应用与页面 |
| `docs/` | 产品与开发文档 |
| `prompts/` | AI Prompt 模板 |

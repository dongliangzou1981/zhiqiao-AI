# 知桥 AI 知识库结构设计（V1.0）

## 概述

知识库以**教材**为根，按**年级 → 学期 → 章节 → 知识点**逐级组织，与课堂教学结构一致，支撑 AI 教案生成、知识点讲解、学情分析与学生答疑等能力。

## 层级结构

```
教材
 └── 年级
      └── 学期
           └── 章节
                └── 知识点
```

| 层级 | 说明 | 示例 |
|------|------|------|
| 教材 | 学科 + 版本 + 学段 | 人教版 · 初中数学 |
| 年级 | 所属年级 | 七年级 |
| 学期 | 上册 / 下册 | 上册 |
| 章节 | 教材单元或章 | 第二章 有理数 |
| 知识点 | 最小教学与检索单元 | 有理数的加减法 |

## 知识点编码规则（V1）

格式：`{学段}-{学科}-{版本}-{年级}{学期}-{章序号}-{知识点序号}`

| 字段 | 含义 | 示例 |
|------|------|------|
| 学段 | `J` 初中 / `P` 小学 / `H` 高中 | J |
| 学科 | 学科缩写 | MATH |
| 版本 | 教材版本缩写 | RJ（人教版） |
| 年级学期 | 两位年级 + 一位学期（1 上 / 2 下） | 71（七年级上册） |
| 章序号 | 两位章节序号 | 02 |
| 知识点序号 | 两位知识点序号 | 03 |

**示例编码**：`J-MATH-RJ-71-02-03` → 人教版初中数学七年级上册 · 第二章 · 第 3 个知识点

## 数据关联（V1 规划）

| 关联对象 | 与知识点的关系 |
|----------|----------------|
| 题目 | 多对一，按知识点打标签 |
| 教学案例 | 多对一，教案/课件引用知识点 |
| 学习事件 | 学生行为记录关联知识点编码 |
| AI Prompt | 输入参数包含知识点编码与名称 |

## AI 服务对接

知识库内容注入 Prompt 后，由 **DeepSeek API** 生成教案、讲解与答疑文本。

| 配置项 | 值 |
|--------|-----|
| Provider | DeepSeek |
| Base URL | `https://api.deepseek.com` |
| 默认模型 | `deepseek-v4-flash` |
| 高阶模型 | `deepseek-v4-pro` |

| 场景 | 建议模型 | 知识库输入 |
|------|----------|------------|
| 学生答疑 | `deepseek-v4-flash` | 当前知识点 `code`、`name`、`description` |
| 知识点讲解 | `deepseek-v4-flash` | 同上 + 章节名称 |
| AI 教案生成 | `deepseek-v4-pro` | 学科、年级、章节、知识点编码与名称 |

> **说明**：DeepSeek API 兼容 OpenAI SDK，项目可使用 OpenAI 兼容客户端配置（`baseURL` + `apiKey`），与 [开发计划](./development-plan.md) 中 AI 封装方式一致。

本地种子数据见仓库 `data/junior-math.ts`，与本文编码规则对齐。
